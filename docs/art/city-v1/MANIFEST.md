Raw 512² masters for the #427 city rework, picked interactively by the operator from
DreamShaper 8 contact sheets. **Not yet game-ready**: backgrounds are baked in (cutouts
tracked separately) and files live here, not in `apps/web/public/art/`.

Generation: local AUTOMATIC1111 (`~/aop-ai-tools/stable-diffusion-webui`, torch pinned
2.3.1 — see the guide's MPS warning), model `DreamShaper_8_pruned.safetensors`, sampler
DPM++ 2M / Karras, 32 steps, CFG 7.5, 512×512 (wall segments 640×384). `gen_city_art.py`
(committed here) holds the shared style prompt + per-building subjects; per-asset seeds are
below. Assets marked _edited_ had manual PIL compositing / inpaint passes on top, so a seed
re-run reproduces only their base image.
